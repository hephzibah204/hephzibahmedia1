## SEARCH INTENT ANALYSIS FRAMEWORK

### Intent Categories

#### INFORMATIONAL INTENT
Keywords showing users want to LEARN, not buy

**Examples:**
- "how to design a website"
- "web design best practices"
- "why is SEO important"
- "what is digital marketing"

**Target:** Blog posts, guides, tutorials  
**CTA:** "Subscribe to newsletter" or "Download guide"

---

#### COMMERCIAL INTENT
Keywords showing users are RESEARCHING services

**Examples:**
- "web design services Nigeria"
- "digital marketing agency near me"
- "best SEO company Nigeria"
- "affordable web design"

**Target:** Service pages  
**CTA:** "Get quote" or "Schedule call"

---

#### TRANSACTIONAL INTENT
Keywords showing users are READY TO BUY

**Examples:**
- "hire web designer Nigeria"
- "book web design service"
- "get web design quote online"
- "start web design project"

**Target:** Service pages + contact forms  
**CTA:** "Start project" or "Chat on WhatsApp"

---

#### NAVIGATIONAL INTENT
Keywords where users are looking for a SPECIFIC company

**Examples:**
- "Hephzibah Media"
- "Hephzibah Media services"
- "Hephzibah Media Nigeria"

**Target:** Homepage, service pages  
**Strategy:** Optimize for brand keywords

## PAGE-KEYWORD MAPPING STRATEGY

These are your main, authoritative pages. They rank for primary keywords.

```
Pillar Page: web-design.html
├─ Primary Keyword: "web design Nigeria"
```

## CONTENT WRITING GUIDELINES FOR SEO

### Heading Structure (Critical)
```
H1: Main Page Topic (1 per page)
"Web Design That Converts – Nigeria's Premium Web Design Agency"

H2: Section Topics (3-5 per page)
- "Why Your Website Design Matters for Nigerian Businesses"
- "Our Web Design Process"
- "Web Design Services We Offer"
- "Portfolio & Case Studies"

H3: Subsections (as needed)
- Specific features, benefits, process steps
```

**Rule:** Each page should have ONE H1, that includes the primary keyword

---

### Keyword Placement Strategy
- **H1:** Include primary keyword (naturally)
- **H2:** Include 2-3 secondary keywords
- **First 100 words:** Include primary keyword
- **Body copy:** Natural keyword placement (1-2% density)
- **Meta description:** Include primary keyword
- **URL:** Include keyword (if appropriate)
- **Image alt text:** Use keyword-related terms

---

### Semantic Keyword Integration
Instead of keyword stuffing, use semantic variations:

**Primary Keyword:** "web design Nigeria"  
**Semantic Variations:**
- website design in Nigeria
- Nigerian web design agency
- professional website design
- digital design services
- UI/UX design services
- responsive design Nigeria

## QUICK KEYWORD RESEARCH PROCESS

**For One Service (Web Design Example):**

1. **Google Keyword Planner** (5 minutes)
   - Search "web design" + filter Nigeria
   - Record: volume, difficulty, variations

2. **Answer the Public** (5 minutes)
   - Search "web design Nigeria"
   - Capture "People also ask" questions

3. **Google Search** (5 minutes)
   - Search primary keyword
   - Note top 5 ranking pages
   - Document page structure

4. **Competitor Check** (5 minutes)
   - Visit top 3 competitor pages
   - Note keywords used in headings
   - Document their structure

5. **Search Console** (5 minutes)
   - Check current rankings (if live)
   - Identify improvement opportunities

**Total time per service: 25 minutes**
