# Hephzibah Media Website Transformation
## Complete Upgrade Strategy & Implementation Roadmap

---

## EXECUTIVE SUMMARY

The Hephzibah Media website will be transformed from a premium informational agency site into a **high-converting sales and lead generation ecosystem**. The existing luxury branding, responsive architecture, and SEO foundation will be retained and enhanced, while adding sophisticated conversion infrastructure, training division, sales funnel services, and intelligent lead capture systems.

**Total Scope:** 6 major phases over 12-16 weeks  
**Technology Stack:** HTML5, CSS3, Vanilla JS, PHP, SQLite (unchanged)  
**Deployment:** cPanel-compatible, lightweight, developer-friendly

CREATE TABLE leads (

### 5.3 Mobile vs Desktop Logic Implementation

#### Mobile Device Detection & WhatsApp Redirect

```javascript
// In form submission handler
function detectDevice() {
    const userAgent = navigator.userAgent;
    const isMobile = /Android|webOS|iPhone|iPad|iPod|BlackBerry|IEMobile|Opera Mini/i.test(userAgent);
    return isMobile;
}

function submitForm(event) {
    event.preventDefault();
    
    const formData = new FormData(event.target);
    const isMobile = detectDevice();
    
    if (isMobile) {
        // Redirect to WhatsApp
        const name = formData.get('full_name');
        const service = formData.get('requested_service');
        const budget = formData.get('budget');
        
        const message = `Hello Hephzibah Media, my name is ${name}. I need ${service}. My budget is ${budget}.`;
        const whatsappUrl = `https://wa.me/2349077780156?text=${encodeURIComponent(message)}`;
        
        window.location.href = whatsappUrl;
    } else {
        // Submit to backend
        fetch('/admin/api/submit-lead.php', {
            method: 'POST',
            body: formData
        })
        .then(response => response.json())
        .then(data => {
            if (data.success) {
                showSuccessMessage();
                setTimeout(() => window.location.href = '/contact.html', 2000);
            }
        });
    }
}
```
